# 반복문(iteration) vs 재귀(recursion)

|              |                  반복문                  |                                          재귀함수                                          |
| :----------: | :--------------------------------------: | :----------------------------------------------------------------------------------------: |
|     정의     |     동일한 코드 블록을 여러 번 실행      |                       함수가 자기 자신을 호출하여 반복되는 작업 수행                       |
|    메모리    |              메모리 힙 사용              | 함수 호출 시 함수의 매개변수, 지역변수, 리턴값, 종료 후 돌아가는 위치가 스택 메모리에 저장 |
| 무한 반복 시 | 무한 루프는 CPU 사이클을 반복적으로 사용 |                              무한 재귀는 스택 오버플로우 발생                              |
|     속도     |                빠른 실행                 |                                         느린 실행                                          |
